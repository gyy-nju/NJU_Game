# NJU_Game

南大校园题材 2D RPG，正在开发中。

## 已实现功能
- 开始画面（标题居中，按空格开始）
- 40×30 图块大地图，四周封闭，包含图书馆、教学楼、食堂、宿舍
- 玩家方向键移动，与建筑/墙壁碰撞检测
- 摄像机跟随，地图边界限制
- 靠近建筑半格时屏幕上方弹出“按 E 进入 XX”提示
- 按 E 进入建筑内部，显示建筑名，底部对话框显示对话内容
- 按空格/E 推进对话，对话结束后自动返回大地图，位置保留
- 中文字体支持

## 开发计划
- 开始界面升级（点击按钮、全屏立绘选人）
- 时间系统（白天/夜晚，建筑外观变化）
- 建筑内部扩展（背景图、NPC 对话、小游戏入口）
- 确认框 UI（进入建筑前二次确认）
- 存档/读档
